# Integrated Real-Time Public Transport and Air Quality Data Platform for Abu Dhabi

## Overview
This platform integrates real-time public transport data with air quality index (AQI) readings for Abu Dhabi, allowing users to make informed travel decisions based on current transport schedules and environmental conditions.

## Features
- Real-time updates of bus routes and schedules
- Current AQI readings across different locations in Abu Dhabi
- Interactive maps showing transport and AQI data
- Social sharing options for users to share travel plans or alerts
- Feedback mechanisms for continuous improvement

## Getting Started

### Prerequisites
- Python 3.x
- Requests library
- Matplotlib library

### Installation
1. Clone the repository from GitHub:
   bash
   git clone https://github.com/your-repo/transport-aqi-platform.git
   
2. Navigate to the project directory:
   bash
   cd transport-aqi-platform
   
3. Install the required libraries:
   bash
   pip install -r requirements.txt
   

### Running the Application
1. Run the Python script:
   bash
   python app.py
   
2. The application will fetch data from the APIs and display it on an interactive map.
